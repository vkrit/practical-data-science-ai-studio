# Sample Data

These synthetic CSV files support the Metropolitan Electricity Authority oriented workshops in the Altair AI Studio course.

| File | Workshops | Main task | Target / output |
|---|---:|---|---|
| `electricity_outage_complaints.csv` | 1, 2, 10, 10B | first process, EDA, text classification, LLM enrichment | `category`, `target_team`, urgency fields |
| `electricity_asset_maintenance.csv` | 3, 4, 6, 11 | preparation, classification, Auto Model, explainability | `failure_next_90d` |
| `new_assets_scoring.csv` | 12 | batch scoring | prediction and confidence |
| `electricity_load_demand.csv` | 5, 9 | regression and forecasting | `peak_load_mw`, `overload_flag` |
| `electricity_service_area_profile.csv` | 7 | clustering / segmentation | service area segment profile |
| `electricity_meter_usage.csv` | 8 | anomaly detection | anomaly score / meter review queue |
| `electricity_recommendation_events.csv` | optional demo | recommendation / next-best-action | `accepted_flag`, `satisfaction_rating` |

Course repository: https://github.com/vkrit/practical-data-science-ai-studio/
